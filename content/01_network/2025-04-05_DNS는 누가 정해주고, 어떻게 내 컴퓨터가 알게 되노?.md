# 2025-04-05_DNS는 누가 정해주고, 어떻게 내 컴퓨터가 알게 되노?

## ✅ 먼저 결론부터 말하자면

> **내 컴퓨터의 DNS 서버는 보통 "자동으로" 할당된다!**

그걸 정해주는 주체는 바로...  
**니가 인터넷 연결할 때 접속한 장비!** 즉,

### 보통은 → **공유기**

### 회사나 기관은 → **DHCP 서버**

### 직접 설정한 경우 → **수동 DNS 서버**

---

## 🧠 구체적으로 설명해줄게

### 1. 니가 와이파이에 연결하거나 랜선을 꽂았다

그럼 어떻게 되나?

→ 니 컴퓨터는 제일 먼저 **"IP 주소", "서브넷 마스크", "게이트웨이", "DNS 서버"** 이 네 가지 정보를 받아야 인터넷 할 수 있거든.

그걸 주는 놈이 바로...

> **DHCP 서버** (Dynamic Host Configuration Protocol)

---

### 2. DHCP 서버가 있는 곳?

- 집에서는 → **공유기가 DHCP 서버 역할**을 함
    
- 회사에서는 → **전용 DHCP 서버 or 라우터**

이 DHCP가 말해주는 거야:

> "야~ 니 IP는 192.168.0.23 쓰고~  
> 게이트웨이는 192.168.0.1이야~  
> DNS는 192.168.0.1(나 공유기)로 써~"

---

### 3. 공유기가 DNS 서버냐고?

> ❌ 직접 DNS 서버는 아님  
> 보통은 **"중간 중계"** 역할을 한다!

즉,  
공유기가 너한테는 DNS처럼 보이지만,  
자기는 또 KT, SK, LG유플러스 같은 통신사의 **진짜 DNS 서버**에다가 묻는 거다.

---

### 4. 그럼 DNS 서버는 어디에 있노?

DNS 서버는 그냥 **전 세계 곳곳에 분산돼 있는 컴퓨터들**이다.

예:

- 구글 DNS: 전 세계 데이터센터에 8.8.8.8 서버 깔려있음
    
- KT DNS: 한국에만 있음
    
- Cloudflare 1.1.1.1: 전 세계에 있음

> DNS 서버는 아무나 만들 수 있어.  
> 다만 빠르고 안정적이고, 전 세계에 퍼져 있으려면 돈이 좀 깨진다 ㅋㅋ

---

### 5. 그럼 난 DNS 서버 어떻게 바꾸노?

니 컴퓨터나 공유기에서 **수동으로 설정**할 수 있다!

#### ✅ macOS에서 바꾸는 법

1. 시스템 설정 > 네트워크
    
2. 와이파이 > 고급 > DNS
    
3. `8.8.8.8`, `1.1.1.1` 이런 거 수동으로 추가 가능!

바꾸면 **더 빠르고 정확하게** 주소를 찾을 수 있는 경우도 많다!
